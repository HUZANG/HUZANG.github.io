我的博客搭建过程与思路整理

简介

为什么想到搭建博客

想要搭建博客有几个目的

  1. 最近点亮了翻墙技能，在Google上能搜索到很多博客，所以我自己也想建一个
  2. 想有一个可以上传自己写的东西的地方

博客的几种种类

  • 独立博客
    • 拥有自己的独立域名和服务器
    • 需要付费和维护复杂
    • 优点是自由度更高
    • 可以类比为自己买的房产
  • 第三方平台博客
    • 外观和布局会被平台局限
    • 技术难度稍低一点
    • 免费使用
    • 可以类比为出租房
  • 自己开发的博客程序
    • 大佬专享

搭建博客的几种途径

  1. 第三方框架+github
    • hexo
    • jekyll
    • hugo
    • vuepress
    • solo
    • wordpress
  2. 第三方博客平台直接注册
    1. wordpress
    2. 博客园
    3. segmentFault
    4. 简书
    5. 掘金
    6. 知乎专栏
    7. Github Page

思路

我在查看了一些教程之后，选择使用hexo博客框架+github来搭建我的博客。这个方案的优点是：

  • 完全免费
  • 技术难度比较小
  • 是目前最主流的搭建方案，因此能在网上找到很多教程和问题的解决方案
  • hexo使用的时间已经很长，可以在github上找到大量的模板
