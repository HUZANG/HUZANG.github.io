《深度工作》读书随笔

书籍信息

  • 作者：【美】卡尔·纽波特 出版社：后浪出版社 出版时间：2017年

一、为什么需要深度工作

互联网和手机的出现的信息爆炸，导致许多人的注意力分散，工作和思考变的碎片化。我发现自己已经很久没有持续的思考或者创造一个东西，逐渐的发展成所谓的心智残疾。

然而我们处在高速发展的时代，带时每刻的巨大变化，代表我们越来越需要快速的掌握一个复杂工具的能力，以及提高工作质量和速度。

以上的两种需求的实现依赖于你的深度工作能力，深度工作可以帮助你快速的掌握困难的事务，和产出一些有价值的东西。

高质量工作产出=时间 X 专注度

深度工作在现在已经越来越少见，所以锻炼和坚持这种能力，可以帮助你成为可以产出精英及工作内容的那批人。

二、深度工作的方法和准则

准则1 工作要深入

如果没有一定的标准和规则，用潜意识去思考或处理工作，我们都会遵循最小阻力原则。所有的困难和障碍都来自欲望。如果我打算完成一件工作，在没有标准和框架的时候，潜意识一定是优先选择最轻松简单的方式，这就算最小阻力原则。但是如果总是习惯用这种方法去完成一件事，所达到的结果也一点是最低的水准。

进行深度工作的状态是困难的，我们必须要有一段时间摒弃所有干扰，来专注的学习和工作，所以必须建立一套自己的时间管理哲学，作者总结了一下几种类型：

1、禁欲型：这种方法适合需要长期进行创作的人，比如作家，或者画家，指在一段长期的时间里，切断与外界联系，包括互联网和社交关系。专注的进行创作和思考。

2、双峰主义：大多数人的工作和环境不允许极端的完全断绝社交的情况下，我们可以在一年里规划一段时间处理需要深度工作的事情，比如比尔盖茨每年有一个思考月，他会在湖边的别墅住一个月，没有只进行读书和思考公司的未来规划。

3、节奏型：在需要专心完成一个项目或者论文的时候，在每天规划出一段90分钟的时间，进行一段深度工作。这需要一些意志力和坚持。当养成习惯之后这是一个适合大多数人的方法。

4、记者型：指类似战地记者一样，能随时进入深度的工作状态，即使在炮火之中，也能专注的进行写作。但我认为这种方法需要多年的专注工作经验，实施难度很高。

对于那些用头脑工作的人来说，他们很少会打乱自己的习惯，将专注工作习惯化，反复锻炼。深度工作最终会变得自然而然。

克里斯坦森认为在职业世界，做什么和如何做的界限往往被忽略，找出某个目标往往相对简单，但真正如何实现的战略才是真正的关键。所以我们需要一种商业的执行模式。

先做最重要的事

4DX原则说，你想做的事越多，真正完成的事反而越少。执行需要把力量集中到“极端重要的目标”上

这样有助于简化选择和集合所有的资源和精力来实现成果。

抓住引领性指标

确定了极端重要的目标后，在4DX框架下，有两种衡量指标，滞后性指标和引领性指标。引领性指标引导你将注意力转移到提升你在短期内可以直接控制的行为上，并对长期目标带来积极性影响。

简单来说，就是多做一些能直观改变你日常状态并且可以推动长期目标进行的事。

准备一个积分板

在团队和个人工作中，如果有一个直观的表格反应出项目和工作的进度和状态，是非常有效的促进执行的方式。

每当你完成一个引领性目标，在积分板上画一个圈，或者一个代表推进进度的曲线，可以更直观的反应你的精力是否用在了最重要的地方。

定期责问

当你建立起了积分板，在一段时间之后，你需要反思哪里做的好或不好。以此确定你是否走在正确的道路上。反省和总结，是商业执行法的重要一环，及时的发现问题和总结经验能够更好的提高效率。

准则2 拥抱无聊

我时常误以为自己能在分心和专注状态切换自如，但是这种想法过于乐观，就像前面说的最小阻力原则。一但你习惯了分心，你就会迷恋于此。在已经习惯分心的现实下，最好的方法是重新编排大脑，让他更适合长期专注任务。

即使你的工作必须大量使用网络和回复信息，那也无妨，假设你必须每15分钟就查看一次信息，平均每次回复5分钟，那么在两个小时的时间段里，每十五分钟安排一次回复时段，就足够了。在两个小时内，你就有90分钟的专注时间。

但是不论你怎么安排你的上网时间，都必须保证在这些时间段外彻底频闭网络。并且在工作之外也按计划使用网络，可以进一步提升专注效果。

要学会拥抱无聊，很多时候以放下手中的事情立刻拿起手机。这时之前的思考和记忆马上被大量的信息替换。实际上人在停止思维活动之后，潜意识依然会在你无聊的时候继续思考。这就是为什么有的人会在工作结束后的下午茶时间灵光乍现。无聊的时间是潜意识继续思考的时间，这非常重要。

准则3 远离社交媒体

许多类似抖音，微博，之类的社交媒体，包含了大量的不必要信息，而且这些媒体的使用机质都是没有终止位置的，这代表你可以一直一直的向下滑动，时间就从这中间溜走了。及时你看到不想看的信息，第一个反应也不是关闭社交媒体，而是划走它。网络上每时每刻都在出现新的信息，于是你甚至可以躺在床上滑动一天的手机。

当然这并不代表着网络社交媒体毫无作用。但你必须在规定的时间使用它，并且设置一个明确的终止时间。
